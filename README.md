# test1_memo

**シンプルなテキストメモ管理アプリ（学習プロジェクト）**

test1_memoは、ソフトウェア開発初心者がWeb開発の基礎を学びながら、実用的なメモ管理アプリを作成する練習プロジェクトです。

## 🌐 デモ

**公開URL**: https://ilucartip.github.io/test1_memo/

ブラウザで今すぐ試せます！データはブラウザのLocalStorageに保存されます。

## 🎯 プロジェクトの目的

1. **学習**: HTML/CSS/JavaScriptの基礎を習得し、Web開発の成功体験を積む
2. **実用性**: 最終的にデスクトップアプリとして日常的に使える状態にする

## ✨ 主な機能

### Phase 1（現在のバージョン）

- ✅ メモの作成・編集・削除
- ✅ フォルダ管理（1階層）
- ✅ **メモのフォルダ移動**（ドロップダウンで簡単に移動）
- ✅ リアルタイム検索（タイトル・本文）
- ✅ 自動保存機能（2秒後に自動保存）
- ✅ キーボードショートカット（Ctrl+S で保存、Ctrl+N で新規メモ）
- ✅ **使い方ガイド**（初回表示時に案内）
- ✅ **成功メッセージ**（操作のフィードバック）
- ✅ **エディタの無効化**（メモ未選択時）
- ✅ LocalStorageでデータ保存

### Phase 2（予定）

- ⏳ デスクトップアプリ化（Electron）
- ⏳ ローカルファイルシステムへのデータ保存
- ⏳ クラウドストレージ連携（OneDrive、Dropbox等）

### Phase 3（予定）

- ⏳ フォルダ階層構造
- ⏳ タグ管理
- ⏳ エクスポート機能
- ⏳ エディタ設定（フォント、サイズ、折り返し）
- ⏳ 編集履歴と差分表示

## 🚀 クイックスタート

### 1. 必要なもの

- Visual Studio Code (VSCode)
- モダンブラウザ（Chrome または Edge）
- Live Server拡張機能（VSCode用）

### 2. セットアップ

詳細は [セットアップガイド](/docs/setup-guide.md) を参照してください。

```bash
# プロジェクトフォルダをVSCodeで開く
1. VSCodeを起動
2. 「ファイル」→「フォルダーを開く」
3. test1_memoフォルダを選択
```

### 3. アプリを起動

```bash
1. src/index.html を VSCodeで開く
2. 右下の「Go Live」ボタンをクリック
```

ブラウザが自動的に開き、test1_memoが表示されます！

詳細は [クイックスタートガイド](/docs/quick-start.md) を参照してください。

## 📝 使い方

### メモを作成

1. 「+ 新規メモ」ボタンをクリック
2. タイトルと本文を入力
3. 「保存」ボタンをクリック（または Ctrl+S / Cmd+S）

### フォルダを作成

1. 左サイドバーの「+ 新規」ボタンをクリック
2. フォルダ名を入力

### メモを検索

1. ヘッダーの検索ボックスにキーワードを入力
2. リアルタイムで検索結果が表示されます

### キーボードショートカット

| ショートカット | 機能 |
|-------------|------|
| Ctrl+S / Cmd+S | メモを保存 |
| Ctrl+N / Cmd+N | 新規メモ作成 |
